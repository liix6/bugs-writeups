# Bugs Writeups

This repository contains real-world vulnerability writeups discovered during penetration testing and bug bounty engagements.

All writeups are based on practical testing against live targets, focusing on real impact rather than theoretical issues.

> Sensitive information such as domains, company names, and tokens has been anonymized.

## Writeups Structure

Each vulnerability writeup includes:

- Summary  
- Vulnerability Details  
- Steps to Reproduce  
- Proof of Concept (PoC)  
- Impact  
- Root Cause  
- Remediation  

## Notes

- All findings were reported responsibly.
- Some details may be modified to avoid disclosing sensitive information.
- The goal of this repository is to share knowledge and improve security awareness.
